# 🎮 Выживи до ОГЭ

Браузерная мини-игра в стиле [Онлайн-колледж №1](https://onlinecollege-1.ru/)

## Как играть

- **Пробел** / **↑** / **тап по экрану** — прыжок
- Уворачивайся от летящих учебников и контрольных
- Собирай золотые звёзды (+50 очков)
- 3 жизни — используй их мудро
- В финале — результат профориентации!

## Технологии

- Чистый HTML5 + Canvas API
- Без зависимостей
- Адаптивный дизайн (desktop + mobile)

## Запуск локально

```bash
npx serve .
# или
python3 -m http.server 8080
```

Открой [http://localhost:8080](http://localhost:8080)
